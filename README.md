# Float Style Guide

Based upon [Foundation for Sites 6](http://foundation.zurb.com/sites)

## Installation


```bash
git clone https://github.com/the-float-yard/style-guide
```

Then open the folder in your command line, and install the needed dependencies:

```bash
cd style-guide
npm install
bower install
```

Run `npm start` to run the Sass compiler. It will re-run every time you save a Sass file.

Point your browser at [localhost:1234](http://localhost:1234) and enjoy.
